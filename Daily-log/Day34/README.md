## The Linux App Store
- **APT** (Advanced Package Tool) is used to manage software on Ubuntu.

## Key Commands
1. **Update the List:** `sudo apt update` (Refreshes the catalog of available apps).
2. **Install App:** `sudo apt install <package_name>` (e.g., `cmatrix`).
3. **The Matrix:** Installed `cmatrix` to turn the terminal into a hacker screen.

## Shortcuts
- **Ctrl + C**: The universal "Stop" or "Kill" command to exit running programs.
